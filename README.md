# Disaster Rescue (Skills Canada 2026) Score Calculator

## Scoring

Total Score: **100**

### Lobby (6)

- **5** points for delivering Steve to the lobby
- **1** point for flipping the breaker switch first

### Debris Zone (24)

- **1** point per piece ABS pipe in the debris zone (4)
- **1** point per piece pool noodle in the debris zone (4)
- **1** point per piece 2x2 wooden block in the debris zone (4)
- **1** point per piece 14/2 wire in the debris zone (4)
- **1** point per broken ECU part in the debris zone (8)

### Environmental Control Unit (44)

- **2** points per fuse correctly installed in the ECU (6)
- **2** points per circuit board correctly installed in the ECU (6)
- **4** points per transformer correctly installed in the ECU (8)
- **1** point per removed broken ECU component (8)
- **1** point per new ECU component delivered to the ECU zone (8)
- **1** bonus point per ECU component correctly installed _if breaker switch is flipped_ (8)

### Autonomous Zone (26)

- **5** points if the autonomous robot ends in the autonomous zone
- **2** points per supply item delivered to safe zone (8)
- **1** point per delivered supply item placed in the correct orientation, with the tape on top (4)
- **2** points per worker remaining standing in the safe zone (4)
- **4** points for placing the fan on the ECU fan platform with any orientation
- **1** bonus point for placing the fan _if breaker switch is flipped_

## Developing

